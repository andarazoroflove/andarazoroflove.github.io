CDEbian is a 32bit Debian 7.x LiveCD which uses the open source Common Desktop Environment as its window manager. It has not been updated in a few years, but is still completely usable and can be found in .iso format on Google Drive.

I am still using a Hyper-V virtual machine of the original CDEbian release which has been updated through Debian 9 and re-built with newer 2.4.x branch CDE code. CDEbian was originally built and tested on a Gateway Solo 2500 366MHz Pentium-II Celeron with 256mb of RAM. The 0.6-0.8 versions are based on VMWare Fusion and make use of the Refracta installer. All versions boot on nearly any legacy hardware or virtual PC software--as an example of compatibility, I recently booted it on VirtualPC 2004 on the iBook G4 I am editing this site with. As an experiment I did a fresh install of CDEbian on an Acer Centrino-based laptop and upgraded it through 9.x with zero issues. Upgrading to 10.x requires a rewrite of the script I call dtlogin with and my VM is primarily used via VNC so its moot.

While I'm on the subject, andarazoroflove is the only 'official' repository of CDEbian code and iso files.
